# 📊 籌碼訊號判讀指南 - 修正版

## ❌ 之前的錯誤

### 問題：邏輯矛盾
```
錯誤範例：🔥外資大買 | ⚠️外資持續賣超
→ 使用者困惑：到底是買還是賣？
```

### 原因分析
- 同時判斷「當日」和「5日累計」
- 沒有明確區分時間維度
- 當日與5日趨勢相反時，造成矛盾

---

## ✅ 修正後的邏輯

### 判讀順序
1. **優先看「5日累計趨勢」** → 主要趨勢
2. **再看「當日動向」** → 轉折訊號

### 訊號定義

#### 主要趨勢（5日累計）
| 訊號 | 條件 | 意義 |
|------|------|------|
| 🔥外資持續大買 | 5日 > +10,000張 | 強力買超 |
| ✅外資累計買超 | 5日 > +5,000張 | 持續買超 |
| ⚠️外資持續大賣 | 5日 < -10,000張 | 強力賣超 |
| ⚠️外資累計賣超 | 5日 < -5,000張 | 持續賣超 |

#### 轉折訊號（當日）
| 訊號 | 條件 | 意義 |
|------|------|------|
| 💡外資今日轉買 | 當日 > +1,000張 且 5日 < 0 | 可能止跌反轉 |
| ⚠️外資今日轉賣 | 當日 < -1,000張 且 5日 > 0 | 可能見頂回落 |

## 📖 實際案例解析

### 案例1: 中信金 ⭐⭐⭐⭐⭐
```
外資當日: +15,262張
外資5日: +17,706張

📊 訊號: 🔥外資持續大買
✅ 解讀: 外資連續多日大舉買進，趨勢明確
💡 操作: 積極買進
```

### 案例2: 鴻海 ⭐⭐⭐
```
外資當日: +1,113張
外資5日: -18,276張

📊 訊號: ⚠️外資持續大賣 | 💡外資今日轉買
✅ 解讀: 
   - 過去幾天都在賣（5日-18,276張）
   - 但今天突然開始買（當日+1,113張）
   - 可能是止跌訊號，需觀察明天是否持續
💡 操作: 謹慎觀察，等待連續2-3天買超確認轉折
```

### 案例3: 台積電 ⭐⭐
```
外資當日: -2,097張
外資5日: -34,190張

📊 訊號: ⚠️外資持續大賣
❌ 解讀: 外資持續賣壓，趨勢未止
💡 操作: 暫時觀望，等待止跌訊號
```

### 案例4: 日月光 ⭐⭐⭐⭐
```
外資當日: +2,640張
外資5日: +29,282張
投信當日: -844張
投信5日: +1,550張

📊 訊號: 🔥外資持續大買 | ⚠️投信今日轉賣
✅ 解讀:
   - 外資持續大買（主力）
   - 投信今天才開始賣（次要）
   - 以外資為主要參考
💡 操作: 仍可持有，外資買盤較強
```

## 🎯 如何運用籌碼訊號

### 最強組合（積極買進）
```
🔥外資持續大買
+ 技術面強勢（K>50, RSI>50）
+ 券資比<5%
= 強力買進訊號
```
**範例**: 中信金、日月光

### 轉折訊號（謹慎觀察）
```
⚠️外資持續大賣 | 💡外資今日轉買
= 可能止跌，需等待確認
```
**操作**: 
- 等待連續2-3天轉買確認
- 搭配技術面支撐位確認
- 不急於進場

**範例**: 鴻海、緯創

### 持續賣壓（暫時觀望）
```
⚠️外資持續大賣
+ 技術面走弱
= 觀望為主
```
**範例**: 台積電、台達電

### 法人分歧（小心謹慎）
```
🔥外資大買 | ⚠️投信大賣
或
⚠️外資大賣 | ✅投信大買
= 法人看法分歧，風險較高
```
**操作**: 以主力法人（通常是外資）為主要參考

## ⚠️ 重要提醒

### 1. 時間維度很重要
- **當日**: 短期波動，可能是雜訊
- **5日**: 短期趨勢
- **1個月**: 中期趨勢

### 2. 轉折需要確認
- 「今日轉買」≠ 立即買進
- 需要連續2-3天確認
- 搭配技術面和量價

### 3. 外資 vs 投信
- **外資**: 資金大，通常較重要
- **投信**: 內資，有時具領先指標
- 兩者一致 > 分歧

### 4. 籌碼只是參考
- 不能單看籌碼
- 要結合技術面、基本面、融資融券
- 綜合判斷才準確
