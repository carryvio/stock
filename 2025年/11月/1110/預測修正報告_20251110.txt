## 二、四大致命錯誤分析

### ❌ 致命錯誤 #1: 過度害怕"RSI超買"

**案例: 華邦電(2344)**
- **我的預測**: "超買回檔，減碼/觀望" ❌
- **實際結果**: +7.59% (軋空行情) 
- **損失**: 錯過大漲+建議錯誤

**錯誤原因**:
1. 只看RSI>80超買，沒看券資比21%的軋空潛力
2. 忽略外資連續大買+22,518張(5日累計)
3. 沒注意美股MU+6.46%、WDC+6.91%強勢連動
4. 機械式套用"RSI超買就看空"的錯誤邏輯

**正確做法**:
```
RSI超買 + 券資比>15% + 外資大買 + 美股同類股>5% 
= 軋空行情，應該看多！
```

## 四、10大修正原則（必須記憶）

### 🔴 絕對原則（違反必錯）

#### 1. 美股連動優先原則
```
當美股同類股大漲(>5%)時，台股相關個股絕對不能看空！
```

**實例**: 華邦電應該看多，因為MU+6.46%

#### 2. 軋空識別原則
```
RSI超買 + 券資比>15% + 外資買超 = 軋空行情，應看多！
```

**實例**: 華邦電 RSI>80 + 券資比21% + 外資大買 = 軋空

#### 3. 投信大賣警示
```
投信5日累計 < -5,000張 = 嚴重警訊，立即降低評分
即使技術面再強也要降評！
```

**實例**: 緯創投信5日-10,638張應該警示

#### 4. AI題材特殊處理
```
AI相關股(創意、世芯)技術面完美多排時，預設看多
除非美股NVDA崩跌
```

#### 5. 記憶體股特殊處理
```
跟隨美股MU走勢，MU大漲時不能看空台股記憶體
```

---

### 🟡 重要修正（經常犯錯）

#### 6. 過度保守問題
```
評分6.5以上不要說"中性"，應該說"偏多"或"積極布局"
```

#### 7. RSI超買不是絕對
```
RSI>70只是警示，還要看:
✅ 法人是否持續買超
✅ 美股是否強勢  
✅ 是否有軋空潛力
```

#### 8. 技術面完美不等於中性
```
完美多排 + MACD多頭 = 應該看多，不是中性！
```

#### 9. 短線/長線區分
```
記憶體、AI股要區分:
- 短線過熱(1-2天) vs 中期趨勢(1-2週)
不能因為短線漲多就全面看空
```

#### 10. 籌碼面優先於技術面
```
當投信大賣時，即使技術面強，也要降評分
籌碼面 > 技術面
```
